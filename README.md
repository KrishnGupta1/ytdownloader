# ytdownloader
A free YouTube video and audio downloader built with Flask and yt-dlp.
A free, lifetime YouTube video and audio downloader built with Flask and yt-dlp. Supports all qualities (1080p, 4K, MP3).

Features
🎬 Download videos in MP4 format
🎵 Download audio in MP3 format
🚀 Supports all video qualities
📱 Mobile-friendly interface
☁️ Deployed on Render (Free Tier)
How to Use
Enter YouTube video URL
Select format (Video/Audio)
Click Download!
Tech Stack
Backend: Flask, yt-dlp
Frontend: HTML, CSS
Deployment: Render
